Channel Attribution Dashboard

Weekly attribution analysis

Mục đích: Phân tích đa kênh (multi-touch attribution) để hiểu channel nào thực sự đóng góp vào conversion, từ đó tối ưu phân bổ ngân sách.

  • Attribution Models: So sánh 4 models (Last-touch, First-touch, Multi-touch, Time-decay)
  • LTV-Based Optimization: Phân bổ ngân sách dựa trên LTV/CAC ratio của từng channel
  • Budget Recommendations: Đề xuất tăng/giảm ngân sách cho từng channel dựa trên data
  • Journey Analysis: Visualize user journey qua các channels

Ra quyết định từ dashboard này:

1. Attribution Insights: Multi-touch model cho thấy Twitch đóng góp 32% revenue (không phải 42% như last-touch).

2. Budget Reallocation: OpenSea có LTV/CAC 69x (cao nhất) nhưng chỉ nhận 30% attribution → Có thể tăng investment.

3. Channel Synergy: Twitter → Twitch → Discord là journey phổ biến nhất (32%) → Tối ưu cross-channel experience.

4. Viral Potential: Discord có viral coefficient 0.52 (cao nhất) → Đầu tư vào referral program.

Acquisition Funnel Dashboard

Daily conversion optimization

Mục đích: Tối ưu từng bước trong acquisition funnel, xác định điểm tắc nghẽn (bottlenecks), và chạy A/B tests để cải thiện conversion rate.

  • Funnel Visualization: 5-step funnel từ Impression → Activated Wallet
  • Drop-off Heatmap: Xem drop-off rate theo từng kênh ở từng bước
  • Segment Analysis: Funnel performance theo user segment (Players, Traders, Collectors)
  • A/B Test Results: Hiển thị kết quả tests với statistical significance

Ra quyết định từ dashboard này:

1. Bottleneck Identification: Step 2→3 (Landing → Connect) có 72.9% drop-off → Ưu tiên fix wallet connection issues.

2. Segment Optimization: Traders có conversion thấp nhất (0.17%) → Cần segment-specific optimization.

3. A/B Test Implementation: Current landing page design (Variant A) có CTR 22% vs Variant B 19.8% → Giữ variant hiện tại.

4. Technical Improvements: 32% users gặp wallet connection errors → Implement auto-retry và better error messages.

5. Revenue Impact Calculation: Fix bottleneck ở step 2→3 có thể tăng revenue $42K/month.
